GUI: To play Tic Tac Toe run the MainGUI.py script
COMMAND-LINE: To play Tic Tac Toe run the Main.py script

This is an implementation of the minimax algorithm and the Monte Carlo tree search.

It might take a while to load the computer's moves when you run the script or launch the GUI. Be patient!

Try to play against the AI, it's very smart for just a bunch of 1's and 0's!

COMMAND-LINE: To make a move, type in the numbers 0 - 8. Each number corresponds to a tile:
0 1 2
3 4 5
6 7 8

GUI: If the game ends in a tie, click either X or O in the bottom left corner to reset the board

COMMAND-LINE: Type -2 to change players
GUI: Click either X or O in the bottom left corner to change players
